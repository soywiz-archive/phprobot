El servidor Master es el encargado de loguear al usuario en todos sus Character servers.

Comunicación: La comuniación con el Master Server es muy sencilla. El Cliente envía el paquete 0x0064 y el servidor responde con 0x0069 si los datos son correctos o con 0x006a en caso contrario.

El servidor tiene una serie de paquetes asociados:

  • Paquetes Recibidos
    • 0x0069 - Login Correcto - Información
    • 0x006a - Login Incorrecto
  • Paquetes Enviados
    • 0x0064 - Login Master Server
