El servidor Character es el encargado de mostrar los Personajes

El servidor tiene una serie de paquetes asociados:

  • Paquetes Recibidos
    • 0x006b - Pet Talk
    • 0x006c - Error logging into Game Login Server (invalid character specified)
    • 0x006e - Fail to Create New Character
    • 0x006f - Success to Delete Character
    • 0x0071 - Recieved character ID and Map IP from Game Login Server
