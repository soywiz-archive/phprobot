El servidor Zone es el encargado de mover uno o varios Mapas, enviar la información de lo que pasa a todos los Jugadores que estén en ese mapa y reaccionar ante las acciones que realicen.

El servidor tiene una serie de paquetes asociados:

  • Paquetes Recibidos
    • 0x0073 - Enter Map
    • 0x0078 - Unit Exists
    • 0x0079 - Unit Connected
    • 0x007b - Entity Move
    • 0x007f - Server Tick
    • 0x0080 - Unit Lost
    • 0x0081 - Disconnected from Server
    • 0x0087 - You Move
    • 0x0088 - Unit Position
    • 0x008a - Attack Damage/Sit/Stand
    • 0x0095 - Entity Name (1)
    • 0x0097 - Private Message From Player
    • 0x008d - Global Message From Player
    • 0x008e - Global Message
    • 0x009c - Unit Look
    • 0x00a3 - Inventory Equipment List
    • 0x00a4 - Inventory Equipment List
    • 0x00b0 - Update Status Long (1)
    • 0x00b1 - Update Status Long (2)
    • 0x00bd - Player Status Info
    • 0x00c3 - Change Equipment Display
    • 0x00e5 - Trade Request
    • 0x00e7 - Trade Response
    • 0x00ea - Trade Add Items
    • 0x00ec - Trade Ok From
    • 0x00ee - Trade Cancel
    • 0x00f0 - Trade Success
    • 0x00fb - Party List
    • 0x0101 - Party Configuration
    • 0x0106 - Party HP
    • 0x0107 - Party Move
    • 0x010f - Skill List
    • 0x0110 - Skill Use Failed
    • 0x0119 - Unit Status
    • 0x011a - Skill Restore
    • 0x0121 - Cart Status
    • 0x013a - Update Attack Range
    • 0x013d - Update HP/SP
    • 0x013e - Skill Cast
    • 0x0141 - Update Status+Bonus
    • 0x0194 - Entity Name (2)
    • 0x0195 - Player Guild Info
    • 0x0196 - Unit Effects
    • 0x019b - Effect
    • 0x01aa - Pet Talk
    • 0x01d0 - Spirit Spheres (1)
    • 0x01d7 - Update Weapon/Armor Display
    • 0x01d8 - Unit Exists
    • 0x01d9 - Unit Connected
    • 0x01da - Unit Moved
    • 0x01e1 - Spirit Spheres (2)
    • 0x01ee - Inventory Item List (2)
    • 0x016c - Guild Name
    • 0x016d - Guild Member Online Status
    • 0x016f - Guild Notice
  • Paquetes Enviados
    • 0x0096 - Private Message From Player
